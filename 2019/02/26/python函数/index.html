从实参的角度分析：

第一种按位置传参

第二种按关键字传参

第三种混着用

如果写成fly(1,x=1)则会报错，因为参数只能赋值一次

如果写成fly(x=1,2)则会报错，因为按位置传值必须在按关键字传值前面

以上两条是混合实参的原则，不能违反

特殊的实参写法

上面的写法相当于： fly(12,16,14,15,18,ni=fuck,sad=many)
